## Progetto scolastico sullo sviluppo di un programma attraverso l'uso dell'intelligenza artificiale (in questo caso gemini).

L'operato dell'IA ha portato alla nascita di un password manager non convenzionale che funziona attraverso diversi strumenti:
1) Script in linguaggio **python** (con l'ibrerie adibite come Fernet e Cryptography)
2) Database supportato da **SQLITE** (upgradable con POSTGRESQL)
3) Interfaccia a linea di comando userFriendly con **Customtkinder**
4) Una **chiavetta USB** con file di riconoscimento.

Questo è l'assetto finale del nostro progetto che ci permetterà di usufruire di un password manager con generatore di password,
**sicurezza** da raimbow table attack e brute force attack dovuto alla necessità di maggiori iterazione da parte di un malware 
nell'escuzione di uno script maligno e di un file di verifica per accedere al vault effettivo, sistema di **protezione in tempo 
reale** con la funzione lock, prevenzione di acesso attraverso masterpassword e connessione USB, **recovery kit** delle password 
legato al sistema **nuke** contro breccie efficaci...
